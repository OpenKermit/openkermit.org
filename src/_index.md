## Welcome to Open Kermit!

Open Kermit is home to the continuing development of [C-Kermit](ckermit), a portable scripting language (similar in concept to the shell, but with built-in screen-scraping automation like expect and built-in file transfer), an implementation of the Kermit file transfer protocol protocol, [scripting language](scripting), network client and server, serial port interface, [BBS client](doc/bbs), and more.
